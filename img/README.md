# Pasta de Imagens

## Insígnia de Angola

Coloque aqui o arquivo PNG da insígnia de Angola.

O arquivo deve se chamar: `insignia-angola.png`

O caminho configurado em `js/config.js` é: `img/insignia-angola.png`

Se preferir usar uma URL externa, altere o valor de `insigniaAngola` em `js/config.js` para a URL completa.
